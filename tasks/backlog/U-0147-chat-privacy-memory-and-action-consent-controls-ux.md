# U-0147 — Chat Privacy/Memory/Action Consent Controls UX

## Priority
- P2

## Dependencies
- B-0379, B-0381, B-0369

## Goal
사용자가 챗봇의 메모리/개인정보/민감액션 실행 정책을 명확히 제어할 수 있도록 설정 UX를 제공한다.

## Scope
### 1) Privacy controls
- 대화 저장 허용/거부, 자동 삭제 주기 설정
- 민감정보 마스킹 알림 및 정책 링크 표시

### 2) Memory controls
- 개인화 메모리 ON/OFF
- 메모리 즉시 삭제/세션 초기화 액션

### 3) Action consent controls
- 민감 액션 이중확인 기본값 선택
- 실행 전 항상 확인 옵션 제공

### 4) UX consistency
- 설정 변경의 즉시 반영 상태 표시
- 모바일/접근성 고려

## DoD
- 개인정보/메모리 설정 변경 성공률 개선
- 사용자 제어권 체감 향상
- 민감액션 실행 전 동의 흐름 명확화

## Codex Prompt
Design user controls for chat trust and privacy:
- Add settings for conversation retention, memory usage, and action consent.
- Provide clear Korean explanations and immediate feedback on policy changes.
- Ensure mobile-accessible controls for privacy-sensitive features.
